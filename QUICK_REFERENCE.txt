# CompliCore - Quick Reference

## 🚀 Start Everything
```bash
cd <repo-root>   # e.g., cd /path/to/CompliCore
docker-compose up -d
```

## ✅ Verify Running
```bash
docker-compose ps
```

## 📊 Watch Logs (Open in separate terminal)
```bash
docker-compose logs -f
```

## 🔍 View Specific Service
```bash
docker-compose logs -f analysis-agent
docker-compose logs -f template-agent
docker-compose logs -f filtering-agent
```

## 🌐 Access Dashboards
- **Web Dashboard**: http://localhost:8080 (Projects)
- **Log Viewer**: http://localhost:8081 (Dozzle)
- **Redis**: redis-cli -h localhost -p 6379

## 📈 Resource Monitor
```bash
docker stats
```

## 🔄 Restart Service
```bash
docker-compose restart analysis-agent
docker-compose restart template-agent
docker-compose restart filtering-agent
```

## ⛔ Stop All
```bash
docker-compose down
```

## 💾 Backup Redis
```bash
docker-compose exec redis redis-cli BGSAVE
```

## 🔧 Scale Analysis Agent
```bash
docker-compose up -d --scale analysis-agent=3
```

## 📋 Troubleshoot
```bash
docker-compose logs --tail=50      # Last 50 lines
docker system df                   # Disk usage
docker-compose ps -a               # All containers
```
